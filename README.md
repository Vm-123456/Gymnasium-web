## 🛠️ Tech Stack
- Bootstrap 5.3.3 | HTML5 | CSS3 | Vanilla JS
- Google Fonts (Poppins) | Font Awesome 6

## 🎯 Features
- 🌙 Dark Mode (localStorage persistence)
- 🎞️ Bootstrap Carousel (5 slides, auto-slide)
- 📝 Registration Form (Bootstrap validation)
- 📱 Fully Responsive (Mobile-first)
- ✨ Scroll Fade Animations (IntersectionObserver)
- 🎨 Color Hunt Palette (#27374D #526D82 #9DB2BF #DDE6ED)
